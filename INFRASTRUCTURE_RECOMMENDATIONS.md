# Slayer Kid Cuts - Infrastructure & Revenue Recommendations

## Salon Booking & POS Systems (DON'T BUILD - USE THESE)

### Recommended: Square Appointments
**Why:** Best for new salons, free tier available, all-in-one solution

| Feature | Details |
|---------|---------|
| **Cost** | Free for solo, $29/mo for Plus (staff scheduling) |
| **Processing** | 2.6% + $0.10 per transaction |
| **Booking** | Online booking, calendar sync, automated reminders |
| **POS** | Full point-of-sale with inventory tracking |
| **Hardware** | Square Reader ($49), Square Stand ($149) |
| **Website Integration** | Embeddable booking widget |

**Action:** Embed Square booking widget on /book page instead of custom form

---

### Alternative: Vagaro
**Best for:** Salons wanting more salon-specific features

| Feature | Details |
|---------|---------|
| **Cost** | Starting at $24/month |
| **Processing** | 2.75% per transaction |
| **Features** | Salon-specific tools, client management, marketing |
| **Booking** | Online booking with deposits |

---

### Premium Option: Boulevard
**Best for:** Established salons with higher volume

| Feature | Details |
|---------|---------|
| **Cost** | $176+/month |
| **Features** | Advanced analytics, AI scheduling, premium support |
| **Best For** | Multi-location, high-end salons |

## Affiliate Revenue Opportunities

### Children's Hair Products

#### Option 1: Amazon Associates (EASIEST)
- **Commission:** 1-10% depending on category
- **Setup:** Sign up at affiliate-program.amazon.com
- **Products to feature:**
  - Kids shampoo/conditioner
  - Detangling brushes
  - Hair accessories
  - First haircut keepsake kits

**Integration:** Add Amazon affiliate links to /shop page products

#### Option 2: Doba Dropshipping (FULL E-COMMERCE)
- **Website:** doba.com
- **Model:** Dropshipping with API integration
- **Pros:** No inventory, direct shipping
- **Cons:** Lower margins, setup complexity
- **Best For:** When ready for real e-commerce

---

### Autism/Sensory Products

#### National Autism Resources
- **Website:** nationalautismresources.com
- **Products:** Sensory tools, fidgets, weighted items
- **Model:** Wholesale/bulk ordering
- **Contact for:** Partnership/affiliate inquiry

#### Fun & Function
- **Website:** funandfunction.com
